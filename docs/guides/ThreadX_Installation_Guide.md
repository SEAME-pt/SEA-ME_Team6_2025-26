# ThreadX - Installation Guide
  This guide provides comprehensive instructions for installing ThreadX.

-----------------------
# Prerequisites
- **Operating System:** Windows 10/11, macOS 10.14+, or a Linux-based OS (Ubuntu/Debian recommended).
- **Python Version:** Python 3.7 or later.
-----------------------
# STEP 1: Python Installation (If Not Installed)
  If you haven't already installed Python, download it from the **official Python website:** https://www.python.org/downloads/ 
  and follow the installation instructions for your platform.
  - on **Windows**, ensure to check the option to "Add Python to PATH" during the installion process.
  - on **MacOS** and **Linux**, Python should come pre-installed. You can check your version by running:
  ```bash
python --version
```
or
```bash
python3 --version
```
------------------------
